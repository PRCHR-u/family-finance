# Family Finance - Frontend

React-приложение для управления семейным бюджетом и учетом финансов.

## Технологии

- **React 18** - UI библиотека
- **Vite** - Сборщик проектов
- **React Router v6** - Роутинг
- **TanStack Query (React Query)** - Управление состоянием сервера
- **Axios** - HTTP клиент
- **Tailwind CSS** - Стилизация (подключается через CDN в index.html)

## Структура проекта

```
frontend/
├── src/
│   ├── api/
│   │   ├── axios.js          # Настройка axios с интерцепторами
│   │   └── services.js       # API сервисы для всех эндпоинтов
│   ├── components/
│   │   └── Layout.jsx        # Основной layout с навигацией
│   ├── context/
│   │   └── AuthContext.jsx   # Контекст аутентификации
│   ├── pages/
│   │   ├── LoginPage.jsx     # Страница входа
│   │   ├── RegisterPage.jsx  # Страница регистрации
│   │   ├── DashboardPage.jsx # Дашборд с аналитикой
│   │   └── DebtsPage.jsx     # Управление долгами
│   ├── router.jsx            # Конфигурация роутинга
│   ├── main.jsx              # Точка входа приложения
│   └── index.css             # Глобальные стили
├── .env                      # Переменные окружения
├── .env.example              # Пример переменных окружения
├── package.json              # Зависимости и скрипты
└── vite.config.js            # Конфигурация Vite
```

## Установка

```bash
# Установить зависимости
npm install

# Создать файл .env (если не существует)
cp .env.example .env
```

## Разработка

```bash
# Запустить dev-сервер
npm run dev

# Приложение будет доступно по адресу http://localhost:5173
```

## Переменные окружения

Создайте файл `.env` в корне frontend:

```env
VITE_API_URL=http://localhost:8000
```

Где:
- `VITE_API_URL` - URL backend API (по умолчанию http://localhost:8000)

## Функционал

### Аутентификация
- Вход по email/паролю
- Регистрация нового пользователя
- JWT токены с автоматическим обновлением
- Защита роутов (PrivateRoute)
- Разделение прав (user/admin)

### Дашборд
- Сводные карточки (общий долг, на модерации, кредиторов, записей)
- Аналитика долга с выбором периода (неделя/месяц/год)
- Детализация по кредиторам
- Таблица последних записей

### Управление долгами
- Просмотр всех долгов
- Фильтрация по статусу (на модерации/подтвержденные)
- Создание новой записи
- Редактирование существующих
- Удаление записей
- Модерация (только admin): подтверждение/отклонение

### Навигация
- Адаптивное меню с основными разделами
- Индикация роли пользователя
- Быстрый выход из системы

## API Integration

Приложение использует следующие API endpoints:

#### Auth
- `POST /auth/login` - Вход
- `POST /auth/register` - Регистрация
- `GET /auth/me` - Получение текущего пользователя

#### Debts
- `GET /debts` - Список долгов
- `POST /debts` - Создание долга
- `PATCH /debts/:id` - Обновление долга
- `DELETE /debts/:id` - Удаление долга
- `POST /debts/:id/approve` - Подтверждение (admin)
- `POST /debts/:id/reject` - Отклонение (admin)

#### Analytics
- `GET /analytics/debt` - Аналитика долга

#### Справочники
- `GET /creditors` - Список кредиторов

## Особенности реализации

1. **React Query** - Кэширование данных, автоматические рефетчи, оптимистичные обновления
2. **Auth Context** - Централизованное управление состоянием аутентификации
3. **API Services** - Модульная структура для работы с API
4. **Axios Interceptors** - Автоматическое добавление токена и обработка 401 ошибок
5. **Tailwind CSS** - Утилитарные классы для быстрой стилизации
